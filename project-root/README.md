# TopBites

TopBites is a restaurant-themed front-end project built using JavaScript, HTML, and CSS.

## About the Project

This project was developed as part of my front-end learning journey and portfolio practice.

TopBites simulates a modern fast-food and restaurant browsing experience where users can navigate between different restaurant sections and explore menu information from multiple cuisines.

The project includes restaurant pages inspired by:

- Italian cuisine
- Japanese cuisine
- Brazilian cuisine
- Mexican cuisine

This was my fifth front-end project and one of the projects where I was able to apply JavaScript in a more organized and structured way.

The entire project was developed in approximately 10 hours.

## Development Process

A large part of the interface structure was built using HTML and CSS, focusing on layout organization, visual consistency, responsive concepts, and navigation flow.

The JavaScript logic was separated into different files to organize responsibilities more efficiently.

The main JavaScript structure handles:

- Navigation behavior
- Scroll interactions
- Dynamic navbar effects
- Show-scroll functionality
- Menu interactions
- DOM manipulation
- Dynamic class management

The project also includes a structured JSON-based system used to organize restaurant information.

Instead of creating completely separate logic structures for each restaurant page, restaurant data such as names, menu items, and specific information are centralized inside a JSON file.

This approach helped improve:
- Code organization
- Data structure understanding
- Dynamic content management
- JavaScript and JSON integration
